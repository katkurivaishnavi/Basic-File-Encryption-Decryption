# Basic File Encryption and Decryption

## Description
This project encrypts and decrypts text files using the Caesar Cipher technique.

## Features
- Encrypts text files
- Decrypts encrypted files
- Simple and user-friendly

## Technologies Used
- Python

  ## Files Included
- encrypt_decrypt.py __ Main Python program
- input.txt __ Original input file
- encrypted.txt __ Encrypted output file
- decrypted.txt __ Decrypted output file

## How to Run the Project
1. Place the text you want to encrypt in `input.txt`
2. Open terminal in the project folder
3. Run the command:
   ```bash
   python encrypt_decrypt.py
4. Choose encrypt or decrypt option.
